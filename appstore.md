### Description (4000 chars max)

TinyPDF is a fast, native PDF text extractor for macOS. Open any PDF, see its text alongside the rendered pages, and export to Markdown. Scanned documents? TinyPDF runs OCR automatically. No subscriptions. No uploads. Everything stays on your Mac.

Features:
- Side-by-side text editor and PDF page viewer
- Automatic text extraction from text-based PDFs
- Built-in OCR for scanned and image-based PDFs using Apple's Vision framework
- Page-by-page text with section headers ("# Page N")
- Export extracted text as Markdown (.md)
- Continuous-scroll PDF preview with auto-zoom
- Folder sidebar with directory browsing
- Quick open file finder (Cmd+P)
- Adjustable font size (Cmd+/Cmd-)
- Word wrap toggle (Option+Z)
- Line numbers toggle (Option+L)
- Handles encrypted PDFs (with password)
- Multiple independent windows
- Light and dark mode — follows your system
- Opens .pdf files directly from Finder
- On-device AI assistant (Cmd+K) — ask questions, summarize, explain. Runs via CoreML, no API key required

Built entirely with native macOS technologies — PDFKit for rendering, Vision for OCR. No Electron. No cloud APIs. Your documents never leave your machine.

Part of TinySuite — a family of native macOS micro-tools that each do one thing well. Every app is built around the file, not around itself. No accounts, no cloud, no lock-in. Open your file, understand it, act on it. Learn more at tinysuite.app.

## Notes for Review

TinyPDF is a native macOS PDF text extractor. It reads local PDF files and can export their text content as Markdown. OCR is performed on-device using Apple's Vision framework. The app does not connect to any server, collect any data, or require an account.
